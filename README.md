# ClassCollaborationRepo

This repository  created for an academic task to understand .

## Purpose of This Repo
The main purpose of this repository is to practice:
- Creating a GitHub repository
- Adding collaborators
- Making commits and tracking changes
- Working together on the same file

## About the Project
This is a simple practice project where multiple collaborators edit the same file.
Each collaborator adds their contribution and commits changes with a message.
GitHub keeps a complete history of all changes made by different users.
